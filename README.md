# Blockchain appliquée à un processus éléctoral
  Langage : C
## Description

L'objectif de ce projet est de protéger les données concernant un processus éléctoral : l'anonymat des électeurs et de leur vote ; et de ne les rendre accessibles qu'à certaines personnes.

Pour cela, nous pouvons distinguer trois parties :

### Cryptage des Données
  * Génération de nombres premiers aléatoires
  * Génération d'une paire (clé publique, clé secrète)
  * Utilisation des clés

### Représentation des Données
  * Encapsulation des données dans des tables de hachage
  * Génération d'un vote

### Création du Blockchain et de son Application
  * Création d'une structure arborescente
